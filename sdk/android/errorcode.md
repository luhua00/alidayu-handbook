# 错误码参考

SMSSDK for Android API回调中，当result=SMSSDK.RESULT_ERROR，则data的类型为Throwable；如果这个Throwable的message就存放有json数据，你可以从中读取status的错误码信息。一部分是服务器返回的。还有一部分是SDK本地生成的。
