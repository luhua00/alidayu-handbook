# 开发者入驻 Registration

## 1. 访问Mob官网首页(http://www.mob.com)
- 点击右上角“注册”进行注册

![注册][1]

- 填写注册信息，输入您的邮箱地址，称呼，公司名称，手机号码等

![输入注册信息][2]

- 点击立即注册，完成注册，并跳转到确认页面

![此处输入图片的描述][3]

- 邮件地址验证成功之后我们进行登陆

## 2. 访问Mob官网首页(http://www.mob.com)进行登陆
- 点击右上角“登录”跳转到登陆页面进行登陆

![此处输入图片的描述][4]

- 输入您注册时使用的邮箱地址和密码进行登陆

## 3. 入驻成功，可进入快速开始或进入后台
![此处输入图片的描述][5]
