# 资费-2016

## 1.1 MOB-SMS 收取哪些费用？
1.使用Mob的SMSSDK2.0以上的版本（包括2.0）完全免费；
2.使用Mob的SMSSDK2.0以下的版本，有相应的收费规则：
> a.使用SDK发送中国大陆地区的短信(文本+语音)，每天可免费发送10000条，超出后每条收费**0.05**元。

> b.使用SDK发送中国大陆地区以外的短信，每天免费20条，超出后按照接收方的价格收取费用，请参见[国家价格表]
 
3.使用Mob的**Web-api**发送短信验证码，根据中国大陆以内和以外区分收费； 
4.使用Mob的**自定义短信接口**发送自定义短信只支持中国大陆地区，所以每条自定义短信收费**0.05元**。

## 1.2 MOB-SMS 的收费方式？

暂时只支持支付宝充值。

## 1.3 如何充值？

开发者后台 --> 充值管理 --> 支付宝充值
